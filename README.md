# FAQ Page

A simple FAQ page built using **HTML**, **CSS**, and **JavaScript**, featuring tabs and accordions to help users navigate common questions.

## Demo
https://krunxlbhoir.github.io/FAQ-Page/

## How to run locally
1. Download or clone the repository.
2. Open `index.html` in any web browser.

## Features
- Tabs to switch between different FAQ categories.
- Accordions that open and close smoothly.
- Two accordions are open by default (as per design).
- Responsive layout for all screen sizes.
- Clean, beginner-friendly code using vanilla JavaScript.

## Notes
- Uses basic DOM manipulation for tabs and accordions.
- Includes transitions on accordion open/close.
- No external libraries or frameworks used.
